# 🎓 Skill Training Institute SQL Project

## 📌 Project Overview

This project is a complete relational database management system developed using MySQL Workbench. It simulates the operations of a Skill Training Institute by managing students, trainers, courses, and enrollments.
The project demonstrates SQL concepts ranging from database creation and table relationships to analytical queries that generate meaningful business insights.


## 🎯 Objectives

- Design a relational database for a training institute
- Store and manage student, trainer, course, and enrollment data
- Perform business analysis using SQL queries
- Practice real-world SQL concepts used in Data Analytics


## 🛠️ Tools Used

- MySQL Workbench
- SQL (MySQL)


## 📂 Database Structure

The database consists of the following tables:

### Students
Stores student information including:
- Student ID
- Student Name
- City

### Trainers
Stores trainer details.

### Courses
Contains course information including course names and fees.

### Enrollments
Maintains relationships between students and courses.

The project includes SQL queries to analyze data such as:

- Total number of students
- Total number of trainers
- Total courses available
- Student enrollment details
- Trainer-wise student count
- Revenue generated from course fees
- Course-wise enrollment count
- Average course fee
- City-wise student distribution
- Highest revenue-generating courses
- Aggregate reports using SQL functions
